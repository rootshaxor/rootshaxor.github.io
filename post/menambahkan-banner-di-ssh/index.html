Menambahkan Banner Di SSH

Erwin Kurniawan

November 21, 2017


A.Pendahuluan
  • Banneradalah sebuah bendera berukuran panjang yang menampilkan sebuah simbol, logo, slogan atau pesan lainnya.
B.Latar Belakang
  • Ini Hanya merubah tampilan login ssh saja , supaya lebih menarik untuk di pandang
C.Maksud & Tujuan
  • Menambahkan Banner di login SSH
D.Alat Dan Bahan
  • Server SSH
  • Cowsay
  • PC Client
  • Jaringan
E.Langkah Langkah
  1. Install Cowsay dengan perintah
    apt-get install cowsay
  2. Kemudian ketikan perintah

    cowsay -f eyes "SMKN 1 DLANGGU" > smk
  3. Selanjutnya pindah smk ke /etc/ssh
  4. Edit file /etc/ssh/sshd_config dan edit seperti ini
  5. Kemudian restart service ssh 
  6. exit dan remote lagi
