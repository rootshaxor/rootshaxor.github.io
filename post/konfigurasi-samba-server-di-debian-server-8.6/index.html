A.Pendahuluan
  • Samba adalah implementasi ulang perangkat lunak bebas dari protokol jaringan SMB / CIFS , dan pada awalnya dikembangkan oleh Andrew Tridgell . Samba menyediakan layanan file dan cetak untuk berbagai klien Microsoft Windows dan dapat berintegrasi dengan domain Microsoft Windows Server , baik sebagai Domain Controller (DC) atau sebagai anggota domain. Pada versi 4, mendukung Active Directory dan Microsoft Windows NT domain.

    Samba berjalan pada sebagian besar sistem Unix , OpenVMS dan Unix , seperti Linux , Solaris , AIX dan varian BSD , termasuk server macOS Apple, dan klien MacOS ( Mac OS X 10.2 dan yang lebih baru). Samba adalah standar untuk hampir semua distribusi Linux dan biasanya disertakan sebagai layanan sistem dasar pada sistem operasi berbasis Unix lainnya. Samba dilepaskan di bawah persyaratan dari GNU General Public License . Nama Samba berasal dari SMB ( Server Message Block ), nama protokol standar yang digunakan oleh sistem berkas jaringan Microsoft Windows.

E.Langkah langkah
  1. Install Terlebih dahulu Sambanya
  2. Jika Sudah selesai edit Config di /etc/samba/smb.conf
  3. Edit config seperti gambar di bawah
  4. Selanjutnya buat samba password untuk user yang di buat login nanti
  5. kemudian restart Sambanya , dengan perintah service samba restart
  6. buat beberapa file untuk mengisi sambanya
  7. kemudian buka di filemanager , pilih connect to server & ketikan smb://ipaddress
  8. masukan username dan password yang telah dibuat
  9. samba sudah berhasil di buat
