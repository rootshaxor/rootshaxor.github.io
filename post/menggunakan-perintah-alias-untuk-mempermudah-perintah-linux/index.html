Menggunakan Perintah Alias Untuk Mempermudah Perintah Linux

Erwin Kurniawan

November 27, 2017

A.Pendahuluan
  • Alias adalah penyamaran nama , di linux  perintah ini dapat di gunakan untuk memepersingkat perintah linux yang panjang , tapi sifatnya untuk sementara
B.Latar Belakang 
  • Karena ada perintah linux yang sangat panjang dalam penggunaan , jadi saya menggunakan perintah alias untuk mempermudah
C.Maksud & Tujuan
  • Mempersingkat perintah linux
D.Alat Dan Bahan
  • Os Linux
E.Langkah Langkah
  1.  Untuk Melakukan Hanya Perlu mengetikan  seperti contoh di bawah
    alias aku="echo -n 'Anda Adalah :'; whoami"
  2. Tapi Di atas hanya untuk Sementara , jika mesin mati dan di hidupkan lagi maka tidak akan bekerja , supaya bekerja terus maka di simpan di dalam .bashrc , karena setiap membuka terminal semua perintah yang di .bashrc di eksekusi otomatis , Tambah kan seperti Berikut 
  3. Coba Kita jalankan perintah apt , yang kita edit di .bashrc tadi

    Nah Langsung muncul permintaan password sudo kan
