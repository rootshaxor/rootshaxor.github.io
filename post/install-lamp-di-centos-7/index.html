Install LAMP Di CentOS 7

Erwin Kurniawan

October 26, 2017

A.Pendahuluan
  • CentOS (Community ENTerprise Operating System) adalah sebuah distribusi linux sebagai bentuk dari usaha untuk menyediakan platform komputasi berkelas enterprise yang memiliki kompatibilitas kode biner sepenuhnya dengan kode sumber yang menjadi induknya, Red Hat Enterprise Linux (RHEL).
B.Maksud & Tujuan
  • Menginstall Lamp Server di Centos 7
C.Latar Belakang
  • Ketika saya bertemu dengan sysadmin sysadmin indonesia yang hebat , mereka berkata bahwa server di perusahaan besar menggunakan  CentOS , itu yang membuat saya ingin membuat server dengan centOS dan Menginstall LAMP di CentOS
D.Alat Dan Bahan
  • PC / Laptop
  • Os CentOS
  • Koneksi Internet
E.Tahab Penginstallan
  1. Pastikan repositori sudah terupdate & upgrade , jika belum lakukan perintah berikut
    sudo yum update && sudo yum upgrade
  2. Selanjutnya Install Apachenya , dengan perintah
    sudo yum install httpd
  3. Jika Sudah Terinstall Aktifkan service apachenya dengan perintah
    sudo service httpd start
  4. Kemudian Cek Di Webbrowser dengan url http://localhost apa Sudah Berjalan Atau belum , jika Muncul seperti gambar di bawah ini maka apache sudah terinstall
  5. Selanjutnya Install Database server , saya menggunakan mariadb server
    sudo  yum install mariadb-server
  6. Kemudian Aktifkan service mariadbnya
    sudo systemctl enable mariadb
    sudo systemctl start mariadb
  7. Selanjutnya Jalankan Perintah
    sudo  mysql_secure_installation
  8. Klik enter saja , kemudian Change password root pilih "Y"
  9. Selebihnya Di Sesuaikan
  10. Selanjutnya Install PHP , di CentOS 7 masih menggunakan php 5
    sudo yum install php php-gd php-intl php-xmlrc php-mysql php-cli php-curl php-soap
  11. Kemudian buat file info.php di /var/www/html/
    sudo nano /var/www/html/info.php
  12. Isikan seperti berikut
    <?php
    phpinfo();
    ?>
  13. Untuk Melihat Hasilnya bisa dilihat di web browser dengan url http://localhost/info.php
  14. LAMP Sudah Terinstall
