Apasih linux itu ?

Linux (diucapkan ˈlɪnəks atau /ˈlɪnʊks/) adalah nama yang diberikan kepada sistem operasi komputer bertipe Unix. Linux merupakan salah satu contoh hasil pengembangan perangkat lunak bebas dan sumber terbuka utama. Seperti perangkat lunak bebas dan sumber terbuka lainnya pada umumnya, kode sumber Linux dapat dimodifikasi, digunakan dan didistribusikan kembali secara bebas oleh siapa saja.

Bagaimana Dengan Sejarahnya ?
Nama "Linux" berasal dari nama pembuatnya, yang diperkenalkan tahun 1991 oleh Linus Torvalds. Sistemnya, peralatan sistem dan pustakanya umumnya berasal dari sistem operasi GNU, yang diumumkan tahun 1983 oleh Richard Stallman. Kontribusi GNU adalah dasar dari munculnya nama alternatif GNU/Linux

Contoh
login Di gunakan untuk login user -
cd Di Gunakan Untuk Masuk Direktory cd namedirektory
ls Di gunakan Untuk Melihat isi Direktoru ls /namedirektory
mkdir Digunakan Untuk Membuat Direktorty mkdir folder1
rmdir Digunakan Untuk Menghapus Direktory (kosong) rmdir folder1
rm Digunakan Untuk Menghapus File & Direktory rm file1 , rm -rf folder1 (folder tidak kosong)
mv Di Gunakan Untuk Memindahkan File & Direktory mv file1 /Document , mv folder1 /opt
pwd Di Gunakan Untuk Melihat Posisi User Yang Digunakan pwd
ifconfig Digunakan Melihat IP pada interface ifconfig wlan0 , ifconfig eth0
whoami Digunakan Melihat nama user whoami
date Digunakan Untuk Melihat hari tanggal waktu tahun date
cal Di Gunakan Untuk melihat tanggal cal
echo Digunakan Untuk Print tulisan Di Terminal echo "hello Androidblackhat"

head Menampilkan isi file , namun hanya bagian atas file saja head file.txt
chmod Merubah permision dari Sebuah File chmmod +x namafile.sh , chmod 777 folder2/ -R
