Download Mp4 & Mp3 Dari Youtube Di Terminal Linux

Erwin Kurniawan

December 2, 2017


A.Pendahuluan
  • youtube-dl adalah utilitas baris perintah untuk mendownload video atau mengekstrak file audio dari situs web streaming seperti YouTube, Dailymotion, dan Vimeo. Perangkat lunak ini ditulis menggunakan Python. youtube-dl adalah perangkat lunak domain publik dengan lisensi dan pengabaian lisensi
B.Latar Belakang
  • Karena sebagian tutorial berupa video , maka saya perlu melihatnya , dalam artian harus di download terlebih dahulu supaya tidak menyianyiakan Bandwitch 
C.Langkah langkah 
  1. Install Terlebih Dahulu youtube-dl , dengan perintah
    apt-get install youtube-dl
  2. Untuk Mendowload MP4 / format video lainya hanya tinggal mengetikkan
    • youtube-dl https://linkdariyoutube.com/videonya
  3. Untuk Mendownload MP3 perintah Yang digunakan
    • youtube-dl https://linkdariyoutube.com/jadimp3 --extract-audio --audio-format=mp3
