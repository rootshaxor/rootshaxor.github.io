Install + Konfigurasi MariaDB-Server di freeBSD

Erwin Kurniawan

October 20, 2017


A.Pendahuluan
  • MariaDB merupakan versi pengembangan terbuka dan mandiri dari MySQL. Sejak diakuisisinya MySQL oleh Oracle pada September 2010, Monty Program sebagai penulis awal kode sumber MySQL memisahkan diri dari pengembangan dan membuat versi yang lebih mandiri yakni MariaDB.
B.Latar Belakang
  • Karena Menginstall Aplikasi Web Kita memerlukan Database / Media penyimpanan , maka dari itu saya menggunakan Mariadb Server untuk menjadi penyimpanan database dari aplikasi web yang akan saya install di FreeBSD saya
C.Maksud & Tujuan
  •  Menginstall Dan konfigurasi mariadb server di FreeBSD
D.Hal Hal Yang Di Butuhkan
  • Internet
  • Virtualbox + FreeBSD
E.Langkah langkah
  1.  Pertama pastikan FreeBSD sudah terhubung ke internet
  2. Sekarang install mariadb servernya dengan perintah

    pkg install mariadb100-server
  3. Setelah Terinstall edit /etc/rc.conf dengan perintah
    ee /etc/rc.conf
  4. Tambahkan Seperti Gambar Di Bawah ini
  5. Jangan Lupa Copy File Yang Ada Seperti di Gambar
  6. Kemudian Simpan Dan Start mariadb servernya dengan perintah

    service mysql-server start
  7. Lihat Apakah service sudah berjalan atau belum , dengan perintah

    sockstat -4 -l
    jika ada tulisan mysql , berarti service sudah berjalan
  8. Selanjutnya membuat password untuk root yang akan di gunakan untuk login mysql dengan perintah

    mysqladmin -u root -p 'Isikanpassworddisini'
  9. Kemudian login ke mysql dengan perintah
    mysql -u root -p
  10. Untuk Passwordnya sesuaikan dengan password yang telah dibuat tadi
  11. Jika berhasil login maka akan seperti gambar di bawah ini
