Membuat POP UP ucapan saat Membuka Terminal Linux

Erwin Kurniawan

November 25, 2017


A.Pendahuluan
  • xcowsay adlah program ASCII yang menggunakan gambar sebagai medianya , xcowsay adalah mode gui dari cowsay
B.Alat Bahan
  • Os Linux
  • xcowsay
  • Koneksi
C.Langkah Langkah
  1. Install Terlebih dahulu dengan perintah

    apt-get install xcowsay
  2. Edit .bashrc dan tambahkan seperti ini

    xcowsay -t 1 "Hay $(whoami)" 2>/dev/null
    Keterangan :
    •  -t 1 adalah lama waktu muncul pop up 
    • $(whoami) ini berfungsi memanggil perintah whoami 
    •  2 > /dev/null berguna untuk merapikan tampilan saat membua xcowsay
  3. Kemudian Buka terminal baru
