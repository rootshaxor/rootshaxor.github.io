Mengganti Template Hostpot di Mikrotik

Erwin Kurniawan

September 12, 2017

A. Pendahuluan

  • Login adalah proses masuk ke jaringan komputer dengan memasukkan identitas akun minimal terdiri dari username/akun pengguna dan password untuk mendapatkan hak akses. Antara username dan password keduanya saat digunakan untuk login harus tepat jangan sampai salah ketik, karena keduanya saling terkait dan tidak bisa dipisahkan. Biasanya username/akun pengguna tidak pernah diubah karena merupakan identitas unik, tetapi password/ kata sandi dapat diubah sesuai keperluan untuk menjaga keamanan akun. 
B. Latar Belakang
  • Karena Saya orangnya bosenan , dan gak yang suka itu itu saja , maka tampilan hostpot yang ada di mikrotik saya ganti dengan template yang lain  
C. Maksud dan Tujuan
  • Untuk mempercantik tampilan login page pada mikrotik hotspot login page 
D. Proses/Tahapan
  • Pertama download FIle page-nya dengan mengetikkan perintah
    • sudo apt-get install -y unrar unzip
  • JIka sudah selanjutnya kita drag and drop file,nya ke mikrotik
     
  • Kita tunggu proses,nya
     
  • Selanjutnya kita masuk ke IP>>Hotspot
  • Kemudian di menu Server profile klik 2x hsprof1
     
  • kemudian pilih masukkan file template tadi di html directory>>apply>>ok
     
  • JIka sudah kemudian kita browser ke hotspot kita
  • Sudah berhasil
