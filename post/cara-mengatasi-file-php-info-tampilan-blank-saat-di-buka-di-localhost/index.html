Cara Mengatasi file php info tampilan blank saat di buka di localhost

Erwin Kurniawan

September 18, 2017


A.Pendahuluan
  • PHPINFO adalah sebuah script yang digunakan untuk menampilkan versi dari php tersebut dalam mode gui , dan menampilkan berbagi ekstensi php yang terinstall
B.Maksud & Tujuan
  • Memperbaiki tampilan blank (kosong) saat membuka  script info.php di localhost
C.Latar Belakang
  • Karena setelah membuat script php yang berisi
    <?php
    phpinfo();
    ?>
    dan di beri nama info.php , dan saat dieksekusi di localhost tidak muncul tampilan apapun atau blank , maka saya mencari solusi untuk masalah tersebut
D.Hal Hal yang di butuhkan
  • Koneksi Internet
  • Pc / laptop yang sudah terinstall webserver
E.Langkah langkah
  1. Pastikan Pc / laptop sudah terkoneksi dengan internet
  2. Buka terminal dan ketikkan perintah :
    apt-get install libapache2-mod-php
  3. Selanjutnya Restart Apache2nya dengan perintah
    service apache2 restart
  4. Buka Kembali info.php di localhost
F.Hasil Yang Di Dapat
  • File info.php berhasil dibuka
