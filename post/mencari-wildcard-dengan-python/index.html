Mencari Wildcard Dengan Python

Erwin Kurniawan

Mencari wildcard pada subnet mask ip

Pendahuluan

Mask Wildcard adalah mask bit yang menunjukkan bagian mana dari alamat IP yang tersedia untuk diperiksa. Di Cisco IOS , mereka digunakan di beberapa tempat, misalnya:

  • Untuk menunjukkan ukuran jaringan atau subnet untuk beberapa protokol routing, seperti OSPF .
  • Untuk menunjukkan alamat IP apa yang harus diizinkan atau ditolak dalam daftar kontrol akses (ACL).

Alat Dan Bahan

  • Laptop/PC terinstall python
  • text editor

Tahap Pelaksanaan

  1. Buka terminal linux dan ketikan perintah

    touch wildcard.py

    perintah di atas di gunakan untuk membuat file yang bernama wildcard.py

  2. Kemudian buka file yang telah di buat dengan text editor kesayangan anda.

    nano wildcard.py

  3. Isikan pada file tersebut dengan beberapa baris code berikut :

    #!/usr/bin/python3

from netaddr import *

ip = IPNetwork('192.168.1.1/24')

print(ip.hostmask)

  4. simpan file tersebut dan jalankan di terminal

    • python3 wildcard.py

      or

    • chmod +x wildcard.py && ./wildcard.py

  5. Selesai

Keterangan

  • ip yang digunakan bebas , asalkan menggunakan ipv4 yang valid
  • untuk mencari wildcard dari subnetmask / prefix yang lain tinggal di ganti saja dengan prefix yang di inginkan
