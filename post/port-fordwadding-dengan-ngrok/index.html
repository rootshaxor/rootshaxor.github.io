Port Fordwadding Dengan Ngrok

Erwin Kurniawan

August 16, 2017



A.Pendahuluan
Pengertian : terowongan intropeksi aman ke localhost - bagian client  
Ngrok menciptakan terowongan dari internet publik ke port di mesin lokal Anda. Anda dapat memberikan URL ini kepada siapa pun untuk mengizinkan mereka mencoba situs web yang Anda kembangkan tanpa melakukan penerapan apa pun.
Ini menangkap semua lalu lintas melalui terowongan. Ini menampilkan informasi tentang lalu lintas HTTP untuk pemeriksaan Anda. Permintaan mentah / byte tanggapan, header yang diurai dan data formulir, pengecekan sintaks JSON / XML dan yang lainnya disertakan. Ini juga bisa memutar ulang permintaan.
Secara default, ngrok akan menggunakan ngrok.com sebagai relay pihak ketiga. Layanan ini disediakan tanpa biaya dan tanpa registrasi, namun dimungkinkan untuk mendapatkan fitur tambahan dengan mendaftar di layanan ini (jenis bayar sesuai keinginan Anda). Namun, mungkin saja setup dan penggunaan servernya sendiri.
Paket ini menginstall bagian client ngrok. Ini bisa langsung digunakan dengan layanan ngrok.com atau dengan server anda sendiri jika anda menginstal paket ngrok-server

E.Langkah Langkah
  1. Login dulu di ngrok.io & download ngroknya di ngrok.io 
  2. jika sudah di download maka extract file ngrok-stable-linux-amd64.zip
  3. ubah permissionya dengan cara chmod +x ngrok
  4. pindahkan ngrok ke /usr/bin/
  5. Install authtokennya ngrok authtoken 2nDvXHg4eZ74SiN2fi52j_5f9ooL6NiAffAWDdSkJCh 
  6. Jalankan ngroknya dengan cara ngrok http 80
  7. coba buka http://xxxxx.ngrok.io di browser 
  8. coba buka lewat handphone (android) , dengan menggunkan mobile data
