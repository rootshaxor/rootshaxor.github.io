Keistimewaan Linux Di Banding Yang Lain


selain opensource linux juga dapat mejalankan program aplikasi operasi sistem lainnya misalnya :

Untuk Menjalankan Program tersebut Anda memerlukan aplikasi berikut :


Program Windows :

  • wine (winehq.org)

    Wine “Wine Is Not an Emulator” adalah lapisan kompatibilitas yang mampu menjalankan aplikasi Windows pada beberapa sistem operasi yang sesuai dengan POSIX, seperti Linux, macOS, & BSD. Alih-alih mensimulasikan logika Windows internal seperti mesin virtual atau emulator, Wine menerjemahkan panggilan Windows API menjadi panggilan POSIX on-the-fly, menghilangkan kinerja dan hukuman memori dari metode lain dan memungkinkan Anda untuk mengintegrasikan aplikasi Windows ke desktop Anda.

  • Frontend wine :

    • q4wine
    • Playonlinux

  • Installing

    • Pilih Package sesuai distro anda Download Wine

Program OSX :

  • darling (darlinghq.org)

    Darling adalah lapisan terjemahan yang memungkinkan Anda untuk menjalankan binari macOS yang tidak dimodifikasi di Linux. Pada dasarnya, ini mirip dengan proyek Wine yang terkenal. Pada titik ini, belum menjalankan aplikasi macOS dengan GUI. Untuk informasi lebih lanjut, tinjau Status Proyek. Pengembang selalu dipersilakan untuk bergabung dengan proyek ini - lihatlah Low Hanging Fruit.

  • Installing

    • Visit Building Darling From Source

Program Android :

  • anbox (anbox.io)

    Anbox menempatkan sistem operasi Android ke dalam wadah, akses perangkat abstrak dan mengintegrasikan layanan sistem inti ke dalam sistem GNU / Linux. Setiap aplikasi Android akan diintegrasikan dengan sistem operasi Anda seperti aplikasi asli lainnya. Untuk mencapai tujuan kami, kami menggunakan teknologi Linux standar seperti wadah (LXC) untuk memisahkan sistem operasi Android dari host. Setiap versi Android cocok untuk pendekatan ini dan kami mencoba mengikuti versi terbaru yang tersedia dari Android Open Source Project.
