D.Hal Hal Yang Di Butuhkan

E.Langkah langkah
  1. cek versi python terlebih dahulu , di sini saya menggunakan versi 2.x.x
  2. selanjutnya buat folder baru dan buat file htmlnya
  3. edit file html.nya
  4. Jalankan service python untuk httpnya
    , 1927 adalah port yang digunakan untuk menjalankan service http
  5. buka di browser http://localhost:1927
