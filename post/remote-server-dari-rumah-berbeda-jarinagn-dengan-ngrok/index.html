Remote Server Dari Rumah Berbeda Jarinagn Dengan Ngrok

Erwin Kurniawan

August 17, 2017



A.Pendahuluan
  • Remote Server adalah mengakses / mengendalikan sebuah server tampa menyentuh fisik dari server tersebut dari jarak dekat maupun jarak jauh , menggunakan satu jaringan maupun berbeda jaringan
  • Remote server berbeda jaringan menggunakan ngrok port fordwardding
B.Maksud dan Tujuan
  • Remote Server Dari Jarak Jauh menggunakan Jaringan yang berbeda , dengan melakukan Port Fordwardding
C.Latar Belakang
  • karena server saya berada di tempat kerja , dan harus datang ke tempat kerja untuk meremote server , maka saya menginstall ngrok di server saya , agar saya dapat meremote server dari jarak jauh ataupun dari rumah 
D.Alat Dan Bahan
  • pc / server
  • ngrok
  • jaringan internet
  • device untuk meremote laptop / smartphone
E.Langkah Langkah
  1. Pastikan di server sudah terinstall ngrok dan sudah terdaftar
  2. buka server melalui web interface
  3. setting ngrok dengan perintah ngrok tcp 22
  4. ngrok sudah berjalan , selanjutnya tutup servernya 
  5. coba meremote server dari smartphone dengan jaringan yang berbeda , di sini saya menggunakan hostpot dari smartphone teman saya 

F.Hasil Yang Di Dapat
  • Hasilnya server saya bisa di remote dengan jaringan yang berbeda dengan ngrok
