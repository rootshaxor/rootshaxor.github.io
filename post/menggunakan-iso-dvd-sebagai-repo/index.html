Menggunakan ISO DVD Sebagai Repo Lokal kurang dari 2 Menit

Erwin Kurniawan

January 11, 2019

Membuat iso file dvd debian 9 sebagai repo local kurang dari 2 menit.

Alat Dan Bahan

- debian-dvd-1.iso
- debian-dvd-2.iso
- debian-dvd-3.iso
- Server yang terinstall debian

Langkah Pembuatan

  1. masuk sebagai user root dan buat direktory sebagai mount point iso dvd.

        
mkdir -pv /media/{dvd1,dvd2,dvd3}

  2. Selanjutnya diasumsikan ketiga iso file sudah di pindahkan ke debian server, pada kasus ini saya pindahkan iso debian di direktory /root/.

  3. kemudian edit file /etc/fstab dengan editor kesayangan anda

  4. tambahkan kedalam file sebagai berikut :

    

/root/debian1.iso   /media/dvd1/    udf,iso9660 loop    0   0
    
/root/debian2.iso   /media/dvd2/    udf,iso9660 loop    0   0
    
/root/debian3.iso   /media/dvd3/    udf,iso9660 loop    0   0

  5. kemudian buka /etc/apt/source.list dan comment semua repo yang online dengan di tambahkan # di depan alamat repo, jika sudah tambahkan sebagai berikut :

        
deb [ trusted=yes ] file:///media/dvd1/ contrib main
    
deb [ trusted=yes ] file:///media/dvd2/ contrib main
    
deb [ trusted=yes ] file:///media/dvd3/ contrib main

  6. Selanjutnya reboot server anda, dan jalankan command.

    apt update

Selesai
