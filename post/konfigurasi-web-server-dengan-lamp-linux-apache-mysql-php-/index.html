A.Pendahuluan
    • Pengertian , LAMP adalah istilah yang merupakan singkatan dari Linux, Apache, MySQL dan Perl/PHP/Phyton. Merupakan sebuah paket perangkat lunak bebas yang digunakan untuk menjalankan sebuah aplikasi secara lengkap.
    • Latar belakang : Ingin memiliki web server di Laptop sendiri


      • Komponen LAMP
        1. Linux  , sebagai Sistem operasi
        2. Apace , sebagai web server
        3. Mysql atau Mariadb , sebagai database
        4. php , python , perl 
       
B.Alat Dan Bahan
    • OS Linux
    • Koneksi Internet 
    • Apache2 
    • Mysql / MariaDB
    • Php5 / Php7

C.Jangka Waktu
Hanya 30 menit bila koneksi internet stabil
