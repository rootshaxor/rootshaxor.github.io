Menggunakan Command sudo tanpa harus memasukan password

Masuk Sebagai Super User

Pertama Login menggunakan super user pada terminal
command: sudo su

Konfigurasi

  • Buat sebuah file di dalam directory /etc/sudousers.d
    command: touch /etc/sudousers.d/erwin
    Ubah erwin dengan username anda / nama yang lain

  • Edit file /etc/sudousers.d/erwin dengan editor kesukaan masing masing
    command: nano /etc/sudousers.d/erwin

  • Isikan file /etc/sudousers.d/erwin seperti berikut
    erwin ALL=(ALL) NOPASSWD: ALL
    erwin adalah username pada komputer saya , jadi sesuaikan dengan username anda masing masing
