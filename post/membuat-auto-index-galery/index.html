Membuat Auto Index Gallery Dengan PHP

A.Pendahuluan
  • PHP: Hypertext Preprocessor adalah bahasa skrip yang dapat ditanamkan atau disisipkan ke dalam HTML.PHP banyak dipakai untuk memprogram situs web dinamis. PHP dapat digunakan untuk membangun sebuah CMS.
B.Maksud Dan Tujuan
  • Membuat Gallery di Portal , dengan menggunakan script PHP , tanpa menggunakan CMS
C.Latar Belakang 
  • Karena Banyak CMS gallery , maka rasanya kurang kreatif bila menggunakan CMS , Jadi saya membuat auto index gallery dengan script PHP
D.Hal Yang Di Butuhkan
  • text editor
  • web server
  • gambar
F.Langkah Pembuatan
  1. Buka Text editor pilihan kalian 
  2. Tuliskan Script seperti di bawah ini <?php
    $folder = "galery/";
    $handle = opendir($folder);
    echo '<fieldset cellspacing="2" cellpadding="5">';
    echo '<legend><marquee class="text" direction="left" behaviot="alternate" scrollamount="5"><img src="img/icons/svg/dlanggu.png">  Welcome To Galery SMKN 1 DLANGGU  <img src="img/icons/svg/dlanggu.png"></marquee></legend>';
    echo '<table>';
    echo '<tr>';
    $i = 1;
    $fileGambar = array('png', 'jpg', 'jpeg', 'gif' , 'svg', 'bmp');
    while(false !== ($file = readdir($handle))){
        $fileAndExt = explode('.', $file);
        if(in_array(end($fileAndExt), $fileGambar)){
            echo '<div class="col-xs-12 col-sm-4 col-md-3 col-lg-3">
                  <div class="tile">
                <img class="tile-image" src="galery/'.$file.'" width="200" height="200" title="Galery Gambar : '.$file.'" /><br />
                <a href="galery/'.$file.'" class="btn btn-primary btn-large btn-block">'.$file.'</a>
                 </div>
                </div>';
            if(($i % 4) == 0){
                echo '</tr><tr>';
            }
            $i++;
        }
    }
    echo '</tr>';  
    echo '</table>';
    echo '</fieldset';
    ?>
  3. Simpan file tersebut dengan nama menampilkan.php 
  4. copykan ke web server dan integrasikan dengan portal
  5. jangan lupa membuat folder galery/ 
  6. Anda Juga Dapat Mengedit dan menambahkan gambar / input upload ke galery ini
