Install + Konfigurasi + ekstensi php5 di freeBSD

Erwin Kurniawan

October 21, 2017


A.Pendahuluan
  • PHP: Hypertext Preprocessor adalah bahasa skrip yang dapat ditanamkan atau disisipkan ke dalam HTML. PHP banyak dipakai untuk memprogram situs web dinamis. PHP dapat digunakan untuk membangun sebuah CMS.
B.Latar Belakang
  • Sebelum Menginstall CMS / Aplikasi web , Kita Harus menginstall PHP beserta extensi yang di butuhkan , agar penginstallan CMS / Aplikasi Web Berjalan Dengan Baik
C.Maksud & Tujuan
  • Menginstall PHP + Extensinya di FreeBSD untuk keperluan penginstallan aplikasi web / CMS
D.Hal Hal Yang Di Butuhkan
  • Koneksi Internet
  • Virtualbox + FreeBSD
E.Langkah langkah

  1. Pertama Install php nya , untuk install php ketikan perintah
    pkg install php56
  2. Kedua Install Ekstensi php , untuk menginstall ketikan perintah

    pkg install mod_php56
  3. Selanjutnya edit dan tambahkan index.php di /usr/local/etc/apache24/httpd.conf , untuk mengedit  gunakan perintah
    ee /usr/local/etc/apache24/httpd.conf ,
    dan tambahkan seperti gambar di bawah
  4. Dan jangan lupa tambahkan

    AddType application/x-httpd-php .php
    AddType application/x-httpd-php-source .phps
  5. Jika sudah simpan & , Lanjutkan dengan mengcopy file /usr/local/etc/php.ini-production menjadi  /usr/local/etc/php.ini , dengan perintah

    cp /usr/local/etc/php.ini-production /usr/local/etc/php.ini
  6. Kemudian edit /usr/local/etc/php.ini , Edit di bagian TimeZone dan tambahkan Asia/Jakarta
  7. Selanjutnya membuat file info.php di /usr/local/www/apache24/data , buat filenya langsung menggunakan perintah
    ee /usr/local/www/apache24/data/info.php
    , isikan seperti di bawah

  8. sekarang restart  apache24
    dan buka web browser demgan alamat ip freebsd slash info.php
    http://ipfreebsd/info.php
  9. Jika sudah muncul info dari php di web browser maka berhasil
