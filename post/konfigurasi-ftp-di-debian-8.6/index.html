Konfigurasi FTP di debian 8.6

Erwin Kurniawan

October 27, 2017


A.Pendahuluan
  • Protokol pengiriman berkas (File Transfer Protocol) adalah sebuah protokol Internet yang berjalan di dalam lapisan aplikasi yang merupakan standar untuk pengiriman berkas (file) komputer antar mesin-mesin dalam sebuah Antarjaringan. FTP merupakan salah satu protokol Internet yang paling awal dikembangkan, dan masih digunakan hingga saat ini untuk melakukan pengunduhan (download) dan penggugahan (upload) berkas-berkas komputer antara klien FTP dan server FTP. Sebuah Klien FTP merupakan aplikasi yang dapat mengeluarkan perintah-perintah FTP ke sebuah server FTP, sementara server FTP adalah sebuah Windows Service atau daemon yang berjalan di atas sebuah komputer yang merespons perintah-perintah dari sebuah klien FTP. Perintah-perintah FTP dapat digunakan untuk mengubah direktori, mengubah modus pengiriman antara biner dan ASCII, menggugah berkas komputer ke server FTP, serta mengunduh berkas dari server FTP.
B.Latar Belakang
  • Karena Saya mempunyai data data di server , maka saya membangun server FTP untuk mempermudah mengambil data saya dari server
C.Maksud & Tujuan
  • Membangun server FTP untuk mempermudah mengambil data file dari server
D.Hal Hal Yang Di Butuhkan
  • Laptop (sebagai client)
  • PC / Server 
  • Koneksi internet
  • proftpd
E.Langkah langkah
  1. Pertama install service ftpnya , dengan perintah
    apt-get install proftpd
  2. Tunggu sampai selesai , maka akan muncul seperti di gambar , pilih yang standalone
  3. Selanjutnya konfigurasi ftpnya , letak konfigurasinya berada di /etc/proftpd , untuk editnya ketikan perintah nano /etc/proftpd/proftpd.conf , kemudian ketikan seperti ini , di paling bawah baris file konfigurasinya
    <Anonymous /home/erwin/>
    User       erwin
    </Anonymous>
    . Sesuaikan dengan nama user & directory masing masing
  4. Setelah itu silakan restart proftpdnya ,
    service proftpd restart
  5. Buka di browser dengan alamat ftp://ipaddressanda , maka akan muncul login
    , isikan sesuai dengan yang telah di konfigurasi tadi , untuk passwordnya sama seperti password user yang telah di konfigurasi tadi
