A.Pendahuluan
  • Server HTTP Apache atau Server Web/WWW Apache adalah server web yang dapat dijalankan di banyak sistem operasi (Unix, BSD, Linux, Microsoft Windows dan Novell Netware serta platform lainnya) yang berguna untuk melayani dan memfungsikan situs web. Protokol yang digunakan untuk melayani fasilitas web/www ini menggunakan HTTP.

D.Hal Hal Yang Di Butuhkan

E.Langkah langkah
  1. Update paket paket dengan perintah pkg update 
  2. Install apache dengan perintah pkg install  apache24
  3. Tunggu sampai selesai
    , jika sudah enable kan apache di /etc/rc.conf dengan perintah ee /etc/rc.conf
  4. Tambahkan  Seperti Di Bawah 

    #Apache
    apache24_enable="YES"
    apache24_http_accept_enable="YES"
  5. Selanjutnya Restart apachenya dengan perintah service apache24 restart
  6. Kita Buka alamat ip freebsd di browser 
  7. Web Server sudah berjalan
