Membuat Direktory di dalam Directory Dan di Dalam Directory

Erwin Kurniawan

November 30, 2017

A.Pendahuluan
  • Direcktory adalah tempat untuk menyimpan file , bisa berdasarkan Jenis File ataupun campur .
B.Latar Belakang
  • Karena Ingin Membuat Folder Project Untuk menyimpan file script saya selama belajar bahasa pemrograman dan membedakan tempatnya tapi hanya sekali membuat Directory Langsung Banyak Isinya
C.Alat Dan Bahan
  • Os Linux
D.Langkah Langkah
  1. Buka Terminal Dan ketikan Perintah

    mkdir -pv Project/{golang/{web,gtk,mobile,cli},Bash/{loop,aritmatic,boolean},Gambas/{kalkulator,infosystem,kasir}}
    di Dalam kurung {} mewakili di dalam Directory , setiap directory yang di dalam directory di pisahkan oleh karakter ","
    Keterangan :
    • Project/{golang, ...} adalah membuat Directroy Golang Di Dalam Directory Project
    • Project/{golang/{cli, ...}  adalah membuat Directroy cli Di Dalam Directory Golang yang berada di Directory Project
    • -p adalah Parents
    • -v berfungsi menampilkan Proses pembuatan Directory
  2. Hasilnya
E.Refrensi 
  • man mkdir / mkdir --help
