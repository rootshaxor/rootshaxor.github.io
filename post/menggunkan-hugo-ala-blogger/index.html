#Menggunakan Hugo Ala Blogger dengan forestry (github)

Pendahuluan

forestry adalah CMS yang dapat digunakan sebagai editor Jekyll, Hugo atau VuePress Anda.

Bagaimana menggunakannya

Kasus kali ini saya deploy site saya di github, jadi di asumsikan menggunakan github

  1. Kunjungi https://forestry.io dan lakukan login menggunakan github
  2. Jika sudah melakukan login , klik button add site , pilih Hugo dan klik button NEXT
  3. Selanjutnya pilih github dan klik button NEXT
  4. Kemudian Pilih repository blognya , pilih Branch master dan klik button NEXT

  5. Tambahkan Email anda , proses ini adalah proses terakhir , jika email sudah di tambahkan maka klik button NEXT

  6. Tampilan forestry
