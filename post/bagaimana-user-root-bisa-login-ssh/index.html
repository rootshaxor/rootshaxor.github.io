Bagaimana User root Bisa Login SSH?


A.Pendahuluan
  • Di SSH defaultnya tidak bisa langsung login menggunakan user root
B.Latar Belakang
  • Karena semua konfigurasi di server menggunakan user root jadi saya harus masuk sebagai user root terlebih dahulu , yang sebelumnya menggunakan user biasa , dan itu sangat membosankan jadi saya konfigurasi ssh saya supaya langsung masuk user root 
C.Maksud & Tujuan
  • Membuat SSH agar bisa login langsung dengan user root
D.Alat Dan Bahan
  • Server
  • PC client
  • Jaringan
E.Langkah Langkah 
  1. Buka file /etc/ssh/sshd_config
  2. Edit Seperti apa yang di tandai di gambar
  3. Restart service ssh
  4. Kemudian keluar , dan remote kembali server dengan menggunakan user root
