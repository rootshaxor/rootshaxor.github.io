Pengenalan Port Fordwadding

A.Pendahuluan
  • Pengertian Dalam jaringan komputer , port forwarding atau pemetaan port adalah aplikasi terjemahan alamat jaringan (NAT) yang mengalihkan permintaan komunikasi dari satu kombinasi alamat dan nomor port ke paket lain sementara paket melintasi gateway jaringan, seperti router atau firewall . Teknik ini paling sering digunakan untuk membuat layanan pada host yang berada pada jaringan yang dilindungi atau disamarkan (internal) yang tersedia untuk host di sisi berlawanan dari gateway (jaringan eksternal), dengan memetakan ulang alamat IP tujuan dan nomor port komunikasi ke Sebuah host internal
B.Maksud & Tujuan
  • Mengenal Dan Memahami apa itu port forwadding
C.Tujuan Port Fordwadding
Port forwarding memungkinkan komputer jarak jauh (misalnya komputer di Internet ) untuk terhubung ke komputer atau layanan tertentu di dalam jaringan area lokal pribadi (LAN).  Pada jaringan hunian biasa, node mendapatkan akses internet melalui DSL atau modem kabel yang terhubung ke router atau network address translator (NAT / NAPT). Host di jaringan pribadi terhubung ke switch Ethernet atau berkomunikasi melalui LAN nirkabel . Antarmuka eksternal perangkat NAT dikonfigurasi dengan alamat IP publik. Komputer di belakang router, di sisi lain, tidak terlihat oleh host di Internet karena masing-masing hanya berkomunikasi dengan alamat IP pribadi.
Saat mengkonfigurasi port forwarding, administrator jaringan menyisihkan satu nomor port di gateway untuk penggunaan eksklusif berkomunikasi dengan layanan di jaringan pribadi, yang berada pada host tertentu. Host eksternal harus mengetahui nomor port ini dan alamat gateway untuk berkomunikasi dengan layanan internal jaringan. Seringkali, nomor port layanan Internet yang terkenal, seperti nomor port 80 untuk layanan web (HTTP), digunakan dalam port forwarding, sehingga layanan Internet umum dapat diterapkan pada host dalam jaringan pribadi.
Aplikasi yang umum termasuk yang berikut:
    • Menjalankan server HTTP publik di dalam LAN pribadi
    • Mengijinkan akses Secure Shell ke host di LAN pribadi dari Internet
    • Memungkinkan akses FTP ke host di LAN pribadi dari Internet
    • Menjalankan server permainan yang tersedia untuk umum di dalam LAN pribadi
D.Jenis-Jenis Port Fordwadding
Port forwarding dapat dibagi menjadi beberapa tipe berikut:
  • Local port forwarding
  • Remote port forwarding
  • Dynamic port forwarding
