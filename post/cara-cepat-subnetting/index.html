Cara cepat subnetting

Erwin Kurniawan

October 25, 2017


A.Pendahuluan
  • Subneting adalah pengelompokan jumlah ip , yang di tujukan untuk menghemat penggunaan dari ip tersebut
B.Latar Belakang
  • karena teman teman saya agak sedikit bingung untuk mengitung sebuah subnet dari sebuah ip maka saya  menggunakan ipcalc untuk menghitung ip
C.Maksud & Tujuan
  • Menghitung jumlah subnet dengan cepat dan mudah
D.Hal Hal Yang Di Butuhkan
  • Laptop / PC
  • Ipcalc
  • Koneksi Internet
E.Langkah langkah
  1. Install ipcalc dengan perintah apt-get install ipcalc -y
  2. selanjutnya cara penggunaan hanya menuliskan ip address dan subnetnya , ipcalc 192.168.22.2/29
