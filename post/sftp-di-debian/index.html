A. Pengertian


  • SSH File Transfer Protocol (Secure File Transfer Protocol, atau SFTP) adalah protokol jaringan yang menyediakan akses file , transfer file , dan manajemen file yang aman dalam aliran datanya . SFTP dirancang oleh Internet Engineering Task Force (IETF) sebagai perluasan dari Secure Shell protokol (SSH) versi 2.0 untuk memberikan kemampuan transfer file aman. IETF Draft Internet menyatakan bahwa, meskipun protokol ini dijelaskan dalam konteks protokol SSH-2, itu bisa digunakan di sejumlah aplikasi yang berbeda, seperti transfer aman file melalui Transport Layer Security (TLS) dan transfer manajemen informasi dalam VPN aplikasi.
B. Latar Belakang
