A. Pendahuluan
1. Pengertian
      Routing Information Protocol (RIP) adalah sebuah protokol routing dinamis yang digunakan dalam jaringan LAN (Local Area Network) dan WAN (Wide Area Network). Oleh karena itu protokol ini diklasifikasikan sebagai Interior Gateway Protocol(IGP). Protokol ini menggunakan algoritma Distance-Vector Routing. Pertama kali didefinisikan dalam RFC 1058 (1988). Protokol ini telah dikembangkan beberapa kali, sehingga terciptalah RIP Versi 2 (RFC 2453). Kedua versi ini masih digunakan sampai sekarang, meskipun begitu secara teknis mereka telah dianggap usang oleh teknik-teknik yang lebih maju, seperti Open Shortest Path First (OSPF) dan protokol OSI IS-IS. RIP juga telah diadaptasi untuk digunakan dalam jaringan IPv6, yang dikenal sebagai standar RIPng (RIP Next Generation / RIP generasi berikutnya), yang diterbitkan dalam RFC 2080 (1997).

D. Tahap Pelaksanaan

      Membuat Topologi

 

    Kita Konfigurasi RO_Pusat

   1. Kita Setting IP Address di
Ether 2 = Ke RO_2
Ether 3 = Ke Client
Ether 4 = Ke RO_3




   2. Kemudian Pada menu pilih "Routing" >> "RIP"


   3. Pada "New RIP Network" isikan dengan "IP Network" yang ada di Interface Router kita.
Ether 2 = 10.10.10.0/29
Ether 4 = 20.20.20.0/29
Ether 3 = 192.168.10.0/24



   4. Kita lihat hasil nya di "IP" >> "Routes" maka hasil nya seperti berikut.


   5. Kita ubah IP Address client sesuai dengan IP yang ada pada Router.


Kita Konfigurasi RO_2

   1. Sama seperti "RO_Pusat", pertama kita setting IP Address nya
Ether 2 = Ke Client
Ether 3 = Ke RO_Pusat




   2. Pilih "Routing" >> "RIP"


3. Pada "New RIP Network" isikan dengan "IP Network" yang ada di Interface Router kita.
Ether 3 = 10.10.10.0/29
Ether 2 = 192.168.20.0/24


4. Kita lihat hasil nya di "IP" >> "Routes" maka hasil nya seperti berikut.


   5. Kita ubah IP Address client sesuai dengan IP yang ada pada Router.


Kita Konfigurasi RO_3

1. Sama seperti "RO_Pusat", pertama kita setting IP Address nya
Ether 2 = Ke Client
Ether 3 = Ke RO_Pusat



   2. Pilih "Routing" >> "RIP"


3. Pada "New RIP Network" isikan dengan "IP Network" yang ada di Interface Router kita.
Ether 3 = 20.20.20.0/29
Ether 2 = 192.168.30.0/24


   4. Kita ubah IP Address client sesuai dengan IP yang ada pada Router.


Sekarang kita coba ping, Apakah sudah terhubung
1. Dari "Client RO_Pusat" ke "Clinet RO_2" (192.168.20.2)

2. Dari " Clinet RO_Pusat" ke "Clinet RO_3" (192.168.30.2)


2. Dari "Client RO_2" ke "Client RO_3" (192.168.30.2)

3. Dari "Client RO_2" ke "Client RO_Pusat" (192.168.10.2)
