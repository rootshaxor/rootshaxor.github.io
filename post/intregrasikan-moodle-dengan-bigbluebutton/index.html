A.Pendahuluan
  • BigBlueButton adalah sistem konferensi web open source . Hal ini didasarkan pada sistem operasi GNU / Linux dan berjalan di Ubuntu 16.04. Selain berbagai layanan konferensi web, ia memiliki integrasi untuk banyak sistem pembelajaran dan manajemen konten utama. Rilis terbaru BigBlueButton adalah versi 1.1
